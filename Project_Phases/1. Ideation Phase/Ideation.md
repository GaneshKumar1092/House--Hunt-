# 💡 Phase 1: Ideation Phase — HouseHunt

## Project Title
**HouseHunt** — A Full-Stack MERN House Rent Management System

---

## Problem Statement
Finding and managing rental properties in India is fragmented, inefficient, and lacks transparency. Tenants struggle to find verified listings; landlords have no centralized platform to manage inquiries and bookings.

---

## Proposed Solution
Build a centralized, user-friendly web platform where:
- **Tenants** can search, filter, and book rental properties
- **Landlords** can list and manage their properties
- **Admins** can moderate listings and oversee platform activity

---

## Project Objectives
- Provide a seamless property search experience with filters (location, price, type)
- Enable secure user registration and authentication
- Allow landlords to submit property listings for admin approval
- Implement a booking system with status tracking
- Build separate dashboards for users and administrators

---

## Target Audience
| User Type | Needs |
|-----------|-------|
| Tenant | Search, filter, view, and book rental properties |
| Landlord | List properties, manage bookings, track status |
| Admin | Approve/reject listings, manage platform users |

---

## Key Features Identified
1. User Authentication (JWT-based)
2. Property Listing with image, price, location, type
3. Advanced Search & Filter
4. Booking System
5. User Dashboard
6. Admin Moderation Panel

---

## Technology Choices (Initial)
- **Frontend:** React.js + Bootstrap
- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + bcryptjs
